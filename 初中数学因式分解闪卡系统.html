因式分解定义

将多项式分解成整式乘积的过程称为因式分解。

点击卡片查看定义和实例

因式分解定义

把一个多项式化成几个整式乘积的形式，这种变形叫作因式分解。

a³ - a = a(a + 1)(a - 1)

例如：

  • a³ - a = a(a+1)(a-1)
  • am + bm + cm = m(a+b+c)
  • x² + 2x + 1 = (x+1)²

更多实例：

  • 2x² - 8 = 2(x² - 4) = 2(x+2)(x-2)
  • a² - b² = (a+b)(a-b)
  • x² + 5x + 6 = (x+2)(x+3)
  • 4a² - 9b² = (2a + 3b)(2a - 3b)
  • xy + x + y + 1 = x(y+1) + (y+1) = (x+1)(y+1)

典型例题

99³ - 99 能被100整除吗？

点击卡片查看解题过程

解题过程

99³ - 99
= 99 × 99² - 99 × 1
= 99 × (99² - 1) [提取公因式]
= 99 × (99 + 1)(99 - 1) [平方差公式]
= 99 × 100 × 98
结论：99³ - 99 = 99 × 100 × 98，因此能被100整除

因式分解与整式乘法的关系

因式分解是整式乘法的逆运算：

(a + b)(a - b) = a² - b²

因式分解形式：

a² - b² = (a + b)(a - b)
因式分解：m² - 16
= (m + 4)(m - 4)
因式分解：y² - 6y + 9
= (y - 3)²
因式分解：2x³ + 8x² + 6x
= 2x(x² + 4x + 3) = 2x(x+1)(x+3)

随堂练习

判断哪些是因式分解？

(a + 3)(a - 3) = a² - 9
m² - 4 = (m + 2)(m - 2)
点击卡片查看解答

随堂练习答案

1. (a + 3)(a - 3) = a² - 9
是整式乘法，不是因式分解（左边是乘积形式，右边是多项式）
2. m² - 4 = (m + 2)(m - 2)
是因式分解（左边是多项式，右边是整式乘积）
3. a² - b² + 1 = (a+b)(a-b) + 1
不是因式分解（结果不是纯乘积形式）
4. 2mR + 2mr = 2m(R + r)
是因式分解（提取公因式）
5. x² + 4x + 4 = (x+2)²
是因式分解（完全平方公式）

几何视角理解因式分解：

a² - a = a(a - 1)
如何用几何图形表示这个因式分解？

几何理解

一个边长为a的正方形：

面积 = a²

减去一个1×a的长方形：

a² - a = a(a - 1)
a²
-a
= a(a-1)
分解步骤：
1. 一个边长为a的正方形 (面积: a²)
2. 减去一个宽为1、长为a的长方形 (面积: a)
3. 剩余部分：长=a，宽=a-1的长方形 (面积: a(a-1))
